     REGISTRATION 
-----------------------------------------------------------------
In order to register please do the following
- select Register menu item from (Help) menu
- In the dialog window insert registration code into Serial number field and click Ok button


     DEMO LIMITATIONS
-----------------------------------------------------------------
If you have obtained the demo from Internet or as a part of any Demo or Shareware collection
you can receive the registration code via Online Order page at http://www.softorbits.com/order.html
Unregistred version works 10 days only.
NOTE! You can send the message to Technical Support via Help=>Send Feedback...

    TECHNICAL SUPPORT
--------------------------------------------------------------------

Please while contacting Technical Support provide the following information:

1. Exact name and version of the program. 
   In some cases date of *.EXE file could be required.
   This information could be found in Modified column of Explorer window.

2. Registration details. 

3. Under which OS the program is running?

4. CPU type and RAM volume.

5. Exact error messages, if any.
